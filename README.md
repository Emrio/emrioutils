## Installing

Downloading from NPM

```shell
$ npm i emrioutils --save
```

Then, in JavaScript:

```js
const u = require('emrioutils')
```

or in Typescript/ES6

```ts
import u from 'emrioutils'
```

or

```ts
import u = require('emrioutils')
```
